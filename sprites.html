Ce sunt CSS Sprites?

O CSS Sprite este o imagine unică care conține mai multe iconițe sau grafice mici aranjate side-by-side. Cu CSS, selectăm și afișăm DOAR porțiunea dorită din imagine folosind proprietățile background-position și background-size.

Avantaje: Un singur request HTTP pentru toate iconițele → pagina se încarcă mai rapid. Browser-ul cach-ează o singură imagine în loc de zeci.

Calculul offset-urilor CSS

Formula: background-position: -(n × dimensiune_afișare)px 0
Exemplu la 16×16px: iconița 1 (Films) → -(1 × 16)px = -16px
